# bugger记录
## 首屏滑动受限
+ 在溢出滑动的情况下首屏向上滑动受限只有线向下滑动后才能向上滑动
+ bugger原因，触顶或触底后滑动还会一直持续
+ 通过触顶或触底+滑动内容的双重判断解决
+ 解决bug心得：主要问题集中在内容溢出时的滑动处理和内容非溢出的滑动解决上，需要对两种滑动状态的值进行深度处理细化
